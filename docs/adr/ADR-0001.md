# ADR-0001: Repository Structure and Build System

**Status**: Accepted
**Date**: 2026-05-06

## Context
Meridian Stream requires a reproducible, benchmarkable, and locally-deployable development environment.

## Decision
Adopt services/ layout with Makefile build system, Go primary language, Docker Compose infra, and GitHub Actions CI.

## Consequences
- Standardized layout clarifies where code belongs
- Makefile provides consistent developer experience
- ADR structure captures architectural decisions

## Alternatives Considered
- Monorepo single cmd/: rejected because project needs multiple services
